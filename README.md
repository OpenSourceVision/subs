# subs

## 简介

本工具用于自动拉取、去重、归类和命名多协议网络代理节点（支持 VMess、VLESS、Trojan、SS、Hysteria 等），并输出整理后的节点列表，方便科学上网用户统一管理节点资源。

## 主要功能

- 支持多种主流代理协议
- 批量拉取并自动解析订阅
- 智能去重（含 VLESS UUID 维度）
- 节点地理归类与自动命名（国旗+地区+序号）
- 多线程并发处理，速率限制与重试
- 详细日志输出

## 使用方法

1. 安装依赖
   ```bash
   pip install -r requirements.txt
   ```

2. 配置 `config.yaml`，填写订阅链接等参数

3. 运行
   ```bash
   python main.py
   ```

4. 结果输出至 `Node.txt`

## 适用场景

- 节点订阅批量管理与去重
- 节点地理分组与命名
- 科学上网节点自动化处理
